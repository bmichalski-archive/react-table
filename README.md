#react-table
A table implementation using React.

##Features
* Pagination
  * Page size selector
  * "Go to page" component
    * Quickly go to distant page

## Works
* In the **browser**
  * See example file example/test.html
* As **CommonJS module**
  * require dist/umd/*.js files
* As **AMD module**
  * ```require('@bmichalski/react-table')```
