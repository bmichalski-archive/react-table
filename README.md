#react-table
A table implementation using React.

##Features
* Filtering
* Pagination
  * "Go to page" component
    * Quickly go to any page
  * Page size selector

## Works
* In the **browser**
  * See example file example/example.html
* As **AMD module**
  * ```require('@bmichalski/react-table')```
